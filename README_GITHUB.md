#### 1️⃣ **Aplicación Web de Gestión de Inventario y Trazabilidad mediante Códigos QR**
* **Descripción**: Plataforma Full-Stack orientada a la trazabilidad de activos corporativos en todas las sedes con almacén de la empresa. Cuenta con una interfaz intuitiva con React/Vite/Tailwind que gestiona la lectura de códigos QR.
* **Backend & Seguridad**: API REST robusta en Java 21 y Spring Boot con seguridad perimetral JWT y Spring Security.
* **Impacto**: Carga masiva e importación automatizada de 15.000 registros desde el ERP/CRM (ADV) a MySQL. Sistema de permisos adaptado para comerciales, técnicos y supervisores.

#### 2️⃣ **SitoInformatic | Plataforma eCommerce y Configurador de Hardware**

* **Motor Heurístico (SitoIA)**: Algoritmo propietario desarrollado en Java 21 que construye la configuración ideal en base a presupuesto y uso final, validando compatibilidad física de sockets, equilibrio térmico (TDP) y balanceo de costes (carryover).
* **Tecnologías**: React, Spring Boot 3.4, PostgreSQL, autenticación stateless (JWT/BCrypt) e integridad transaccional ACID.

#### 3️⃣ **Desarrollador de Soluciones de Automatización / Analista de Datos**
* **Descripción**: Single Page Application enfocada en la automatización contable de registros de clientes.
* **Algoritmos**: Motor de emparejamiento heurístico y diccionario de pesos de palabras clave para normalizar textos, clasificando automáticamente más del 80% de registros.
* **Funcionalidad**: Importación/exportación con parser JSON, soporte Drag & Drop e integración híbrida con Inteligencia Artificial. Generador automático de scripts de migración SQL idempotentes por provincias.

#### 4️⃣ **Asistente Virtual de Recomendación Inteligente (DMS/ECM)**
